# 全局调控：

全局调控指的是渐进化的改变，类似定投的Portfolio，并不是立刻买入一大笔，然后就坐视不管。而是轻微改变定投金额，逐渐增加/减少。每一笔定投看上去都微不足道，无法直接影响全局，但极其多的小金额定投将汇聚成一笔可观的定投计划。

在这种背景下，单次的错误决策并不会对全局造成影响，就比如，我错误地提升了某笔定投计划当中10%的金额，10天后我纠正错误，它对于全局而言是微不足道的，因为我可以在事情变坏之前调整改变，而这给了我更大的容错空间。	

这就像是在太空中漂流的飞船，只需要很微弱的力，即使是那种比克重还低的单位，只要不断叠加，随着时间，它就一定会飞出太阳系，这就是全局调控的意义。

因此，今年626的目标不在于巨大的改变，不设置任何宏大计划，只做局部的优化，只做微小的改变。
